# Client Agent Guide

## Scope
Applies to files under `client/`.

## Stack
- React
- Vite
- TypeScript
- Tailwind CSS

## Frontend Conventions
1. Feature-first structure under `src/features/*`.
2. Pages in `src/pages`.
3. Reusable UI in `src/components`.
4. API client logic in `src/api`.
5. Keep components small and typed.
6. Prefer controlled forms and explicit validation states.
